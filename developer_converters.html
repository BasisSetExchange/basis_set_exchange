Output converters¶

converters - Conversion basis set output¶

Conversion of basis sets to various formats

bse.converters.write_g94(basis)¶

Converts a basis set to Gaussian format

bse.converters.write_gamess_us(basis)¶

Converts a basis set to GAMESS-US

bse.converters.write_json(basis)¶

Converts a basis set to JSON

bse.converters.write_nwchem(basis)¶

Converts a basis set to NWChem format

refconverters - Conversion of reference data¶

Conversion of references to various formats

bse.refconverters.compact_elements(elements)¶

Create a string (with ranges) given a list of element numbers

For example, [1, 2, 3, 6, 7, 8, 10] will return “H-Li,C-O,Ne”

bse.refconverters.reference_text(ref)¶

Convert a single reference to plain text format

Parameters:
ref : dict

Information about a single reference
bse.refconverters.write_bib(refs)¶

Converts references to bibtex

bse.refconverters.write_json(refs)¶

Converts references to JSON format

bse.refconverters.write_txt(refs)¶

Converts references to plain text format
